## Axchange 使用教程：连接安卓手机的完整步骤

连接之前需要先在手机上开启 USB 调试模式。这是 ADB 工具的要求，不是 Axchange 特有的限制。

**第一步：在安卓手机上开启开发者模式**

进入「设置 > 关于手机」，连续点击「版本号」7次，系统会提示开发者模式已开启。

**第二步：开启 USB 调试**

回到「设置」，找到「开发者选项」，打开「USB 调试」开关。

**第三步：用数据线连接手机和 Mac**

接上数据线后，手机上会弹出一个授权弹窗，询问是否允许这台电脑调试。选择「允许」，勾选「始终允许」省去以后每次确认的麻烦。

**第四步：在 Mac 上打开 Axchange**

Axchange 启动后会自动检测连接的安卓设备，左侧边栏会出现你的手机。点击进去就能浏览整个文件系统了。

**第五步：传文件**

把文件从 Mac 拖进 Axchange 窗口就上传了，把 Axchange 里的文件拖到 Mac 桌面或文件夹就下载了。或者右键菜单里也有传输选项。
